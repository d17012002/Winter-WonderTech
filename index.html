Perks of the Event


In a world driven by technology, creative and interactive websites are the doors through which one expresses their distinctive skills to the people of the world. It is only through the knowledge of HTML and CSS that one can achieve the peak of uniqueness in their websites.

The event spanned over the weekend is packed with fun games, fascinating quizzes, and informative sessions. The “Bit By Bit” coding club brings a great learning experience for aspiring web developers as we unveil the HTML and CSS Workshop event.
